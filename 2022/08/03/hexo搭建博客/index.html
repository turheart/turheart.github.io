绑定域名(非必须内容)

虽然在Internet上可以访问我们的网站，但是网址是GitHub提供的:yourusername.github.io，如果我们想使用自己的个性化域名，这就需要绑定域名。此处演示阿里云的域名绑定。

个人域名获取

域名购买网址:https://cn.aliyun.com (阿里云官网)

域名解析

打开阿里云管理控制台，找到你的个性化域名，进入解析，然后添加解析，包括三条解析记录。

第一条:记录类型:A，主机记录:@，记录值:192.30.252.154

第二条:记录类型:A，主机记录:@，记录值:192.30.252.153

第三条:记录类型:CNAME，主机记录:www，记录值:yourusername.github.io

然后登录GitHub，进入之前创建的仓库，点击settings，设置Custom domain，输入你的域名，点击save保存。

hexo设置

注意

  • 在执行hexo d命令登录GitHub时，需要生成token替换密码登录。token生成步骤如下：
    1. 登录github账号，点击头像选择Settings
    2. 在settings中选择Developer setting
    3. 选择Personal access tokens，然后选中生成令牌Generate new token
    4. 选择repo，最后生成令牌 Generate token
